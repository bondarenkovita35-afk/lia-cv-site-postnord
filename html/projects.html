Huvudprojekt

True Sweden

True Sweden är en egenutvecklad fullstack-plattform som kombinerar e-handelsflöden med AI-stödda funktioner, tydlig systemstruktur och administrationsfunktioner. Projektet har gett mig praktisk erfarenhet av att arbeta med både teknisk struktur och användarnära digitala flöden i samma lösning.

Projektöversikt

Live projekt

I projektet har jag arbetat med planering av struktur, utveckling av frontend och backend, databasintegration, API-lösningar och vidareutveckling av funktioner steg för steg. Fokus har legat på tydliga användarflöden, stabil kod och en lösning som går att bygga vidare på.

Öppna live demo

Vad jag har byggt

  • Sidflöden för produkter, konto, beställning och administration.
  • Backendlogik för datahantering, autentisering och API-baserade funktioner.
  • Frontend med fokus på tydlig navigation, interaktion och användarupplevelse.
  • Funktioner som har förbättrats och byggts ut iterativt över tid.

Teknisk stack

Next.js React TypeScript HTML / CSS REST API Prisma ORM SQL / PostgreSQL Git / GitHub

Vad projektet visar

  • Att jag kan planera och strukturera ett större webbprojekt.
  • Att jag kan arbeta med både frontend, backend, databaser och API:er.
  • Att jag kan utveckla, testa och förbättra en lösning över tid.
  • Att jag förstår sambandet mellan teknik, innehåll och användarflöden.

Hur jag arbetar i projekt

  • Jag delar upp större uppgifter i tydliga steg och arbetar metodiskt tills flöden fungerar stabilt.
  • Jag testar, felsöker och förbättrar kontinuerligt i stället för att lämna lösningar halvdiga.
  • Jag dokumenterar viktiga delar så att nästa steg blir enklare att förstå och bygga vidare på.
  • Jag arbetar med fokus på struktur, ansvar och långsiktig vidareutveckling.
