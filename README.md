LIA Portfolio – POSTNORD

This package contains a single portfolio project prepared for a LIA application.

Contents
- index.html – tailored portfolio landing page
- html/cv.html – CV in web format
- html/personligt-brev.html – cover letter in web format
- html/projects.html – project page with True Sweden and demo link
- pdf/ – PDF versions of CV and personligt brev
- assets/profile.jpg – profile photo

Live demo link included in the project:
https://www.truesweden.co

Start locally
Open index.html in a browser or run a small local server.
